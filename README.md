ROGUER - Sistema de Inventarios y Préstamos

ROGUER es una plataforma integral desarrollada con Django diseñada para la gestión eficiente de activos tecnológicos.
El sistema permite un control total sobre el ciclo de vida de los equipos, desde su ingreso hasta su asignación o suspensión.

Características Principales
Registro y Categorización: Clasificación organizada de nuevos equipos para una búsqueda y control simplificados.
Gestión de Ubicaciones: Asignación dinámica de activos a diversas oficinas o departamentos.
Control de Estado (Bajas): Flujo de trabajo para dar de baja equipos, gestionar su reparación y reactivarlos en el inventario.
Módulo de Préstamos: Registro detallado de préstamos vinculados a usuarios específicos, garantizando la trazabilidad de cada ítem.
 
TecnologíasBackend: Python & Django Web Framework.
Base de Datos: SQLite.
Frontend: CSS, Bootstrap, JS, Chart.js y Django Templates.
